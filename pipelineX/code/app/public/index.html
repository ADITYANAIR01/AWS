How the pipeline works
  1. Code is pushed to GitHub and triggers the build pipeline.
  2. CodeBuild installs dependencies and builds the app.
  3. The Docker image is created and pushed to ECR.
  4. ECS deploys the container and starts the service.
  5. The app becomes live on port 3000 with environment metadata shown above.
